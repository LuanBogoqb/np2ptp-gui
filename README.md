# np2ptp-gui

[![Release](https://github.com/LuanBogoqb/np2ptp-gui/actions/workflows/release.yml/badge.svg)](https://github.com/LuanBogoqb/np2ptp-gui/actions/workflows/release.yml)
[![Latest release](https://img.shields.io/github/v/release/LuanBogoqb/np2ptp-gui)](https://github.com/LuanBogoqb/np2ptp-gui/releases/latest)

A desktop app for [np2ptp](https://github.com/LuanBogoqb/np2ptp). Paste a link, pick a folder, watch it download — no terminal, no flags to remember.

![np2ptp-gui, Downloads tab](docs/assets/screenshot-downloads.png)

## Get it

Download the latest build from [Releases](https://github.com/LuanBogoqb/np2ptp-gui/releases/latest). Nothing to install, no extra runtime to hunt down.
